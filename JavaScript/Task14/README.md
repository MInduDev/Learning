# JavaScript Loops
## Project Description

This project demonstrates the use of **JavaScript loops** and basic programming concepts. The output is displayed in the **browser console**.

The project includes examples of:
- Sum of first n numbers
- Multiplication table
- Prime number check
- Factors of a number
- Sum of digits
- Armstrong number check


## Project Structure

javascript-loops/
│── index.html
│── style.css
│── script.js
└── README.md


## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in your web browser.
4. Press **F12** or **Ctrl + Shift + I** to open Developer Tools.
5. Go to the **Console** tab to view the program output.



## Tasks Performed
### 1. Sum of First n Numbers
Calculates the sum of the first `n` natural numbers using a `for` loop.

**Example Output**
Sum of first 10 numbers = 55


### 2. Multiplication Table
Prints the multiplication table of a given number from 1 to 10.

**Example Output**
5 x 1 = 5
5 x 2 = 10

5 x 10 = 50


### 3. Prime Number Check
Checks whether a given number is prime or not using a loop.

**Example Output**
7 is Prime Number


### 4. Factors of a Number
Displays all factors of a given number.

**Example Output**
Factors of 13:
1
13


### 5. Sum of Digits
Calculates the sum of all digits of a given number using a `while` loop.

**Example Output**
Sum of Digits = 9


### 6. Armstrong Number Check
Checks whether a given number is an Armstrong number.

**Example Output**
153 is an Armstrong Number
