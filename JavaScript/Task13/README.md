# Task 13 - JavaScript Even or Odd
## Objective
This project checks whether a given number is even or odd using JavaScript.

## Files
- index.html
- style.css
- script.js
- README.md

## Features
- Uses four different variables.
- Uses if-else statements.
- Displays the result in the browser console.

## How to Run
1. Open the project folder.
2. Open index.html in a web browser.
3. Press F12 or Right Click → Inspect.
4. Open the Console tab.
5. View the output.

## Sample Output
The number 5 is odd.
The number 10 is even.
The number 17 is odd.
The number 20 is even.
