# Task 19 - Service Booking Web App
## Project Description

This is a simple Service Booking Web Application created using HTML, CSS and JavaScript DOM. Users can browse different laundry services, add them to the cart, remove items, and complete the booking using the booking form.

## Features

- Responsive Navigation Bar
- Browse 6 Services
- Add Service to Cart
- Remove Service from Cart
- Dynamic Total Amount
- Booking Form Validation
- Local Storage Support
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM)
