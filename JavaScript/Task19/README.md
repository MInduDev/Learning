# Task 19 -  Service Booking Web App
## Project Description

The Service Booking Web App is a simple JavaScript DOM project that allows users to browse different laundry services, add services to the cart, and book them. The application updates the cart dynamically, calculates the total amount, and validates the booking form before confirming the booking.

This project is created using basic HTML, CSS, and JavaScript and is suitable for beginners who are learning DOM manipulation and event handling.

---

## Features

- Responsive Navigation Bar
- Logo and Navigation Links
- Username and Logout Button
- Browse Laundry Services
- Display Service Image, Name, and Price
- Add Item Button
- Skip Item Button
- Dynamic Cart
- Dynamic Total Amount
- Booking Form
- Form Validation
- Success Message after Booking
- Beginner-Friendly JavaScript Code

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## How to Run the Project

1. Download or clone the project.
2. Open the project folder.
3. Make sure all images are inside the **images** folder.
4. Open **index.html** in your web browser.
5. Browse services.
6. Add services to the cart.
7. Fill the booking form.
8. Click the **Book Now** button.
