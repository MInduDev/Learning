# Task 12 - JavaScript Compound Interest Calculator
## Objective

Create a JavaScript program that calculates compound interest using predefined values.

## Formula

A = P × (1 + r / n) ^ (n × t)

Where:

- A = Final Amount
- P = Principal Amount
- r = Annual Interest Rate
- n = Number of times interest is compounded per year
- t = Time in years

## Files

- index.html
- script.js
- README.md

## How to Run

1. Open the project folder.
2. Open `index.html` in your browser.
3. Press **F12** or **Right Click → Inspect → Console**.
4. The calculated result will be displayed in the browser console.

## Sample Output

```
Principal Amount: 40000
Interest Rate: 10%
Time: 3 years
Compound Interest: 13240.00
Final Amount: 53240.00
```
