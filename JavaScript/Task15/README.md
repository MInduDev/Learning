# Task 15 - Array & Functions
## Objective

The objective of this assignment is to create JavaScript functions that receive an array as an argument and perform different operations on it.

## Tasks Performed
This project performs the following operations on a dummy array:

1. Find the maximum number in the array.
2. Calculate the sum of all elements in the array.
3. Count the number of odd numbers in the array.
4. Display the results using `console.log()`.

## Function Types Used
This project uses different types of JavaScript functions:
- Arrow Function
- Anonymous Function
- Normal Function

## Dummy Array
javascript
[4, 8, 2, 11, 6, 7, 10]


## Sample Output
text
Array: [4, 8, 2, 11, 6, 7, 10]
Maximum number: 11
Sum of all elements: 48
Count of odd numbers: 2
