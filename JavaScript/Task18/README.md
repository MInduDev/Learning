# Task 18 - Greeting Web Page using DOM
## Objective

The objective of this assignment is to design a simple web page using HTML, CSS, and JavaScript. The page includes a greeting message, four clickable color boxes, a text input for the user's name, and a button to greet the user.

## Features

- Header displays **Hello** initially.
- Four boxes labeled Red, Blue, Green, and Yellow.
- Input field to enter the user's name.
- Button to greet the user.
- Uses DOM manipulation.
- Changes the greeting to **Hello, [name]** when the button is clicked.
- Changes the background color of each box when it is clicked.
- Simple and responsive layout.

## Files

- `index.html` - Contains HTML, CSS, and JavaScript.
- `README.md` - Project documentation.

## How It Works

### Greeting

1. Enter your name in the input box.
2. Click the **Greet** button.
3. The heading changes from:

```
Hello
```

to

```
Hello, Your Name
```

### Color Boxes

Click any box to fill it with its respective color.

- Red → Red background
- Blue → Blue background
- Green → Green background
- Yellow → Yellow background

## Sample Actions

### Greeting

- Input: Reet
- Output:

```
Hello, Reet
```

### Color Box

- Click **Red**
- Output: Red box turns red.

- Click **Blue**
- Output: Blue box turns blue.

- Click **Green**
- Output: Green box turns green.

- Click **Yellow**
- Output: Yellow box turns yellow.

## How to Run

1. Save the file as `index.html`.
2. Open it in any web browser.
3. Enter your name and click the **Greet** button.
4. Click any color box to change its background color.
