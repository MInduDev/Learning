# THALA 7 - Sneaker Store

A simple CSS Mini Project created using HTML and CSS.

## Project Overview

THALA 7 is a basic sneaker store website that contains multiple pages for showcasing products, shopping cart functionality, and payment details.

## Features

- Responsive Navigation Bar
- Hero Section
- Product Showcase
- Product Gallery
- Shopping Cart Page
- Coupon Code Section
- Order Summary
- Payment/Checkout Page
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
