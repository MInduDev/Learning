# Bella Vista Restaurant Website

A responsive restaurant website built using HTML5 and CSS3. The website showcases an Italian restaurant with sections for Home, Menu, About Us, Contact Information, Reservation Form, and Footer.

## Features

- Responsive Design
- Fixed Navigation Bar
- Hero Section with Background Image
- Menu Display Section
- About Us Section
- Contact Information Section
- Reservation Form
- Footer with Quick Links and Social Media Links
- Smooth Scrolling Navigation

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design (Media Queries)
