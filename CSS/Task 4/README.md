# Task4 - CSS Units, Box Model & Fonts

## Description
This task demonstrates the use of:

- External CSS
- Viewport Units (vw, vh)
- Box Model
- Gradient Background
- Image Styling
- Button Styling
- Proper HTML Structure

## Files Included

1. index.html
2. style.css
3. README.md

## Features

- Equal spacing of 25% from left and right.
- Equal spacing of 10% from top and bottom.
- Gradient styled button.
- Responsive layout using viewport units.
- Uses box-sizing property.
- No Flexbox used.
- No Relative/Absolute positioning used.
- No overflow hidden used.
- Page remains without scrollbars.

## How to Run

1. Open index.html in any web browser.
2. Ensure style.css is present in the same folder.
3. The page will display an image section with a gradient button.
