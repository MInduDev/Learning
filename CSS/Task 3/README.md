# Task3 - CSS Selectors Assignment

## Overview

This project demonstrates the use of CSS selectors such as:

- Element Selector
- Class Selector
- ID Selector
- Descendant Selector
- Pseudo-class Selector

The webpage contains headings, paragraphs, lists and hyperlinks styled according to the assignment requirements.

## Files

### index.html
Contains the webpage structure.

### style.css
Contains all styling rules.

## Features Implemented

1. Linked external CSS file to HTML.
2. Applied background color to the page.
3. Created three heading elements using class="head".
4. Applied dark blue text color using class selector.
5. Added lime background to the second heading using id="bglime".
6. Styled the word "head" in red inside the second heading.
7. Styled all paragraphs using the p selector.
8. Customized the second paragraph using id="para".
9. Increased font size and changed color of the word "paragraph" inside the second paragraph.
10. Styled hyperlink text without using inline CSS, id or class attributes on anchor tags.

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Open the project folder.
2. Open index.html in any web browser.
3. Verify that all styles are loaded correctly from style.css.
