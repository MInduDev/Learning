# Task 7 - Laundry Service Responsive Web Page

## About This Project

This is my Laundry Service landing page project created using HTML and CSS.

The main goal of this task was to learn how to make a webpage responsive using Flexbox and Media Queries.

## Features

* Responsive Navbar
* Responsive Hero Section
* Flexbox Layout
* Mobile Friendly Design
* Tablet Friendly Design
* Hover Effect on CTA Button

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## How to Run

1. Download the project folder.
2. Open the folder in VS Code.
3. Make sure index.html and style.css are in the same folder.
4. Open index.html in your browser.
5. Resize the browser window to test responsiveness.

## What I Learned

While doing this task I learned:

* How Flexbox works
* How Media Queries are used
* How to make layouts responsive
* How to adjust font sizes for different screen sizes
* How to create hover effects using CSS
