A responsive Laundry Service landing page built using HTML and CSS.

- Responsive Navbar
- Responsive Hero Section
- Desktop, Tablet and Mobile Support

- Mobile Navigation Optimization

## Technologies Used

- Flexbox

## Responsive Breakpoints

- Mobile: Below 768px
