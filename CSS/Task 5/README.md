# Task5 - Laundry Services Hero Section

## Project Overview

This project is a simple Laundry Services landing page created using HTML and CSS.

The page contains:

- Navigation Bar
- Hero Section
- Service Heading
- Description Text
- Call To Action Button
- Laundry Related Image

## Files Included

1. index.html
2. style.css
3. README.md

## CSS Concepts Used

- Display Inline-Block
- Width and Height
- Margin and Padding
- Border Radius
- Text Styling
- Viewport Units (vw, vh)

## Assignment Requirements Covered

- Navigation bar with Logo, Home, Services, About Us and Contact Us
- Username section on the right side
- Hero section divided into two parts
- Left side contains heading, description and button
- Right side contains laundry related image
- Viewport units used for responsive sizing
- External CSS file linked with HTML
- No Bootstrap used
- No Flexbox used

## How To Run

1. Open the project folder.
2. Open index.html in a browser.
3. The page will load automatically with all styles from style.css.
