# Task6 - Laundry Mart Login Page

## Overview

This project is a simple login page designed for a Laundry Mart web application.

The layout is divided into two sections:

1. Image Section
2. Login Form Section

The page uses Flexbox to create a clean and centered layout.

## Files Included

- index.html
- index.css
- README.md

## Features

- Split screen login layout
- Centered content using Flexbox
- Responsive design
- Email and Password input fields
- Login button
- Forgot Password link
- Register Now link
- Laundry related illustration

## Technologies Used

- HTML5
- CSS3

## CSS Concepts Practiced

- Flexbox
- justify-content
- align-items
- flex-direction
- box-shadow
- border-radius
- form styling

## How To Run

1. Open the project folder.
2. Open index.html in any browser.
3. Ensure index.css is present in the same folder.
4. The login page will load automatically.

## Notes

- No Bootstrap has been used.
- Styling is done entirely with CSS.
- The container height is set using viewport height (vh).
